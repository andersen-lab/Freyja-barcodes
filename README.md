# Freyja-barcodes

This repository stores current and historic lineage/genotype barcodes and corresponding lineage hierarchy information for pathogens other than SARS-CoV-2, for the [Freyja tool](https://github.com/andersen-lab/Freyja). Barcodes can be directly accessed through Freyja using the `--pathogen` option, or can be separately downloaded and supplied to freyja via the `--barcodes` option. For more information, please refer to the documentation for Freyja ([link](https://andersen-lab.github.io/Freyja/index.html#)).
